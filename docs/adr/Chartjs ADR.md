# ADR: Gebruik van Chart.js voor datavisualisatie

| Name              | Chart.js voor datavisualisatie                                  |
|-------------------|-----------------------------------------------------------------|
| Current version   | 1                                                               |
| Current status    | 🟩 **DECIDED**                                                  |
| Problem/Issue     | Wij hebben een oplossing nodig voor datavisualisatie in onze webapplicatie. De oplossing moet eenvoudig te integreren zijn, aantrekkelijk uitziende grafieken bieden, en geschikt zijn voor verschillende typen visualisaties. |
| Decision          | Gekozen voor Chart.js als bibliotheek voor datavisualisatie.    |
| Alternatives      | 1. D3.js: Zeer krachtig en flexibel, maar vereist meer ontwikkeltijd en heeft een steilere leercurve. </br> 2. Highcharts: Veel functionaliteit en eenvoudig te gebruiken, maar het is een commerciële oplossing met licentiekosten. </br> 3. ApexCharts: Een moderne bibliotheek met vergelijkbare mogelijkheden als Chart.js, maar met minder communityondersteuning. |
| Arguments         | 1. Eenvoudige integratie: Chart.js is eenvoudig te installeren en biedt een intuïtieve API voor snelle integratie in webtoepassingen. </br> 2. Uitgebreide grafiekopties: Ondersteunt diverse typen grafieken zoals lijngrafieken, staafdiagrammen, cirkeldiagrammen en meer, met uitgebreide aanpassingsmogelijkheden. </br> 3. Open source: Chart.js is gratis te gebruiken en heeft geen licentiekosten, wat het een kosteneffectieve keuze maakt. </br> 4. Community en documentatie: Chart.js heeft een actieve community en goede documentatie, wat ondersteuning bij implementatie en probleemoplossing vergemakkelijkt. </br> 5. Responsiviteit: De grafieken zijn standaard responsief, waardoor ze goed werken op verschillende schermformaten. |
